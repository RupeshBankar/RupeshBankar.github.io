About This Page

Here, I will be going over some features of this website that may be useful if someone was grading this as an assessment (wink wink)

Ways I used JavaScript:

  • IntersectionObserver; to detect when navigation bar is past a certain point so that it can change its colour
  • Glider.js; a library that makes making image sliders very simple
  • p5.js; a library to easily draw and process things on a canvas
  • ParentNodes; used to move placeholders to the top in input boxes
  • EventListeners; used for many effects and animations
  • Randomizer; simple randomizer that picks a random number for the "Guess-The-Number" game
  • jQuery; library used to animate elements (on hover)

Ways I used HTML & CSS:

  • Local Variables; used to change the colour of the navigation bar
  • Fixed Position; navigation bar fixed in place
  • ::before & ::after; used for many things, such navigation bar hover, hamburger menu, glitch loading screen effect
  • Forms (+ inputs); used to make login page, contact page, survey page, etc
  • :hover; used on buttons, links, navigation bar
  • :focus; used for effect in forms
  • @keyframes; used for simple animations
  • Display; used block, flex, inline, etc for UI
  • Position; used fixed, absolute, relative to position things correctly
  • Transitions; used for smooth hover and focus effects
  • Translate; used for 3D effects (game menu)
  • @media screen; used only a few times to make things slightly responsive
  • Anchor Tags <a>; for links to other websites/songs (etc)
  • Lists; just like this one!

And plenty more for you to discover!
